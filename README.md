Инициализация проекта:
1. В главной папке запустить команду <b>pnpm i</b>

Запуск проекта: 
1. Запуск всей сборки сразу <b>pnpm dev</b>
2. Запуск фронта <b>pnpm dev w</b>
3. Запуск бэка <b>pnpm dev b</b>
