Инициализация проекта:
1. В главной папке запустить команду <b>pnpm i</b>

Запуск проекта: 
1. Запуск всей сборки сразу <b>pnpm dev</b>
2. Запуск фронта <b>pnpm dev w</b>
3. Запуск бэка <b>pnpm dev b</b>


-------------------------------------- GIT ---------------------------------------

1. Добавить все изменения <b>git add</b>

2. Закоммитить все изменения <b>git commit -m "[комментарий]"</b>

3. Запушить(отправить на сервер) все изменения <b>git push</b>

4. Запулить(получить с сервера) все изменения <b>git pull</b>

5. Переход по веткам <b>git checkout [название ветки]</b>
